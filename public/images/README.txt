איך להוסיף תמונות:
1. הורד תמונות מאתרים כמו:
   - unsplash.com
   - pexels.com
   - pixabay.com

2. שנה את שם הקובץ לאחד מהשמות למעלה

3. העתק את הקובץ לתיקייה הזו

4. התמונה תוצג אוטומטית באתר

הערה:
זוהי דרך זמנית לניהול תמונות.
בפרויקט אמיתי, מומלץ:
- להעלות תמונות לשירות אחסון כמו Cloudinary או AWS S3
- לאפשר העלאת תמונות דרך טופס הוספת מוצר
- לשמור רק את ה-URL של התמונה בדאטהבייס
